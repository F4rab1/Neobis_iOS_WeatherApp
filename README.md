# iOS Weather App

A simple iOS app displaying current weather conditions (temperature, wind, humidity, visibility, air pressure) and a 5-day forecast using the OpenWeather API.

## Tools & Technologies:

- UIKit
- Swift
- URLSession
- StackView
- MVVM (protocol-based)
